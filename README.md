## Summary 
A Portfolio built with express.js.

![](/express.jpg)

### To Run the project on local machine

* In the project directory, you can run:

`npm install`

### Note:Make sure you have node and npm installed

* Then run
 
 `npm start`

* Open a browser on localhost
